快速开始

3 分钟上手 Penshot

📦 安装

pip install penshot

🚀 基础用法

import asyncio
from penshot import PenshotFunction

async def main():
    # 创建智能体
    agent = PenshotFunction()

    # 提交任务并等待结果
    result = agent.breakdown_script("""
        场景：现代办公室
        人物：小李（程序员）
        动作：小李正在写代码，突然接到电话
    """)

    if result.success:
        print(f"生成成功！共 {len(result.data['shots'])} 个镜头")
    else:
        print(f"生成失败: {result.error}")

asyncio.run(main())

🌐 REST API

# 启动服务
penshot serve-rest --port 8000

# 提交任务
curl -X POST http://localhost:8000/api/v1/storyboard \
  -H "Content-Type: application/json" \
  -d '{"script": "你的剧本内容...", "language": "zh"}'

# 查询状态
curl http://localhost:8000/api/v1/status/{task_id}

# 获取结果
curl http://localhost:8000/api/v1/result/{task_id}

API 接口

完整的 REST API 接口文档

📝 POST /api/v1/storyboard

异步提交分镜生成任务

{
  "script": "剧本文本",
  "language": "zh",
  "callback_url": "可选的回调地址"
}

返回: task_id 和状态

🔄 POST /api/v1/storyboard/sync

同步提交，等待完成

{
  "script": "剧本文本",
  "timeout": 300
}

返回: 完整的分镜结果

📊 GET /api/v1/status/{task_id}

查询任务状态

返回: 状态、进度、阶段信息

📄 GET /api/v1/result/{task_id}

获取任务结果

返回: 分镜序列、统计数据

🗑️ DELETE /api/v1/task/{task_id}

取消任务

❤️ GET /api/v1/health

健康检查

返回: 服务状态和统计
